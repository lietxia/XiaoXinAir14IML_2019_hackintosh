mergevars

 

        -mergevars:[ImagePathName] -nvpath:[FactoryCopyPathName] -out:[OutputPath]

 

To merge the data in variable store region and factory copy region into a bios image in the same regions.

The following command is required:

   -nvpath: specify the file path of a file which has variables data in variable store region and factory copy region

 

The following command is optional:

   -out: specify the bios image!|s output file path. If the output file path is not specified, the operation result will be saved to the original one (i.e., [ImagePathName]).
